# BACKEND
## Folder Structure

```
backend\
 |--controllers\    # Route controllers and handle business logic
 |--middlewares\    # Custom express middlewares
 |--managements\    # Route controllers for admin usage
 |--test\           # test function
 |--models\         # Mongoose models 
 |--routes\         # Routes
 |--utils\          # Utility classes and functions
 |--main.js         # Entry point
 ```
